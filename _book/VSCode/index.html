VS Code

VS Code 环境搭建

正所谓工欲善其事必先利其器，搭建一个好的开发环境能大大提升我们的开发效率。

命令行工具

打开 VS Code，打开控制面板（⇧⌘P），输入 shell command，在提示里看到 Shell Command: Install 'code' command in PATH，运行它就可以了。

  • 使用 code . 命令可以在 VS Code 中直接打开当前文件或文件夹

插件

  • LeetCode 随时随地刷题
  • Instant Markdown 本地预览 Markdown 文件
  • Polacode 将指定代码变成图片
